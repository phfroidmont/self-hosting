# NixOS 26.05 Upgrade Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Update the self-hosting flake to NixOS 26.05 and handle required release-note compatibility changes.

**Architecture:** Keep the upgrade targeted: change inputs, replace removed Promtail with Alloy, and fix known evaluation blockers. Avoid service major upgrades beyond what the NixOS release requires.

**Tech Stack:** Nix flakes, NixOS modules, Grafana Alloy, Loki, SOPS, deploy-rs.

## Global Constraints

- Keep `services.postgresql.package = pkgs.postgresql_15`.
- Keep `services.nextcloud.package = pkgs.nextcloud32`.
- Replace `services.promtail` with `services.alloy`.
- Use a SOPS-backed Grafana `security.secret_key`.
- Build/evaluate both `hel1` and `relay1` before claiming completion.
